# Learning Rate Schedule Comparison: Constant vs Cosine vs Warmup+Cosine

> IdeaClaw ARC Pipeline · Real Experiment Execution · First-Hand Data
> Run ID: ic-20260319-lr-schedule

## Abstract

We compare three learning rate schedules (constant, cosine annealing, warmup+cosine) on polynomial regression (y = 3x² - 2x + 1) using a 2-layer network with numerical gradients. **Cosine annealing at lr=0.001 achieves the lowest test loss (38.19), a 1.1% improvement over constant lr (38.60).** Warmup+cosine at the same lr shows a smaller 0.4% improvement. Higher lr (0.005) with warmup+cosine degrades performance by 3.3%, suggesting warmup does not eliminate lr sensitivity.

### Key Findings

1. **Cosine annealing is the best schedule** at lr=0.001 (test loss 38.19 vs 38.60) ✅
2. **Warmup does not help** when base lr is already appropriate (-0.4% vs -1.1%) ✅
3. **5× higher lr with warmup hurts** — warmup cannot compensate for bad base lr (+3.3%) ✅
4. Train-test gap is consistent (~6.4 across runs), no overfitting detected ✅
